# Web Page Skill

This skill creates web pages and applications.

## Usage

When the user asks for:
- Websites
- Web apps
- Landing pages

## Process

1. Analyze requirements
2. Create specification
3. Generate HTML/CSS/JS
4. Verify implementation

## Templates

### Landing Page

```html
<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <header>Header</header>
  <main>Content</main>
  <footer>Footer</footer>
  <script src="app.js"></script>
</body>
</html>
```

### Single Page App

Generate React/Vue/vanilla JS as appropriate.

## Anti-Trope Design

Follow anti-trope principles:
- No purple gradients
- No rounded corners
- Monospace headings
- Sharp, industrial aesthetic

## Output

Save to:
- `/mnt/user-data/workspace/index.html`
- `/mnt/user-data/workspace/style.css`
- `/mnt/user-data/workspace/app.js`

Present files when complete.
