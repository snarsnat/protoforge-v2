# Research Skill

This skill researches topics deeply using available tools.

## Usage

When the user asks to:
- Research a topic
- Find information
- Learn about something

Use web search and fetch tools to gather information.

## Process

1. Use `web_search` to find relevant sources
2. Use `web_fetch` to get detailed content from top sources
3. Synthesize findings into a coherent response

## Best Practices

- Search for multiple perspectives
- Verify information across sources
- Cite sources in responses
- Be thorough but concise
