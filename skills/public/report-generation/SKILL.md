# Report Generation Skill

This skill generates comprehensive reports from research.

## Usage

When the user asks for:
- Reports
- Summaries
- Analysis documents

## Report Structure

```markdown
# Report Title

## Executive Summary
Brief overview of the topic.

## Introduction
Background and context.

## Findings
Main research findings with sources.

## Analysis
Detailed analysis of findings.

## Conclusion
Summary and recommendations.

## References
Cited sources.
```

## Output

Save to:
- `/mnt/user-data/outputs/report.md`
- `/mnt/user-data/outputs/report.pdf` (if PDF generation enabled)

## Best Practices

- Use web search to gather information
- Cite sources properly
- Structure reports logically
- Include visualizations where appropriate
